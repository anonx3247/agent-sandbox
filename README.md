# agent-sandbox

General Agent Sandboxing with Sane Defaults `--yolo` without hassle.

`agent-sandbox` (`asb`) is a general, srt-based sandbox wrapper around any coding
agent (claude, codex, pi, …). It runs your agent under a sane-default sandbox so
you can grant broad autonomy without the usual hassle.

## Usage

```
asb [flags] -- <command...>
```
